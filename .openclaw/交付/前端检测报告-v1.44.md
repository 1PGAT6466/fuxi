### 6.3 CSP 策略分析

```
default-src 'self';
script-src 'self' 'unsafe-hashes' https://d3js.org https://cdn.jsdelivr.net + 多个 sha256;
style-src 'self' 'unsafe-inline';
img-src 'self' data: https:;
connect-src 'self';
frame-ancestors 'none';
```

- ✅ `default-src 'self'` — 严格默认策略
- ✅ `frame-ancestors 'none'` — 防点击劫持
- ✅ `connect-src 'self'` — 限制 API 请求来源
- 🟡 **MEDIUM**: `style-src 'unsafe-inline'` — 允许内联样式，存在样式注入风险（但影响较低）
- 🟡 **MEDIUM**: `script-src` 中的 `'unsafe-hashes'` + 多个固定 sha256 — 维护成本高，建议改用 nonce
