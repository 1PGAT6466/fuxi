# 伏羲 v1.50 修复脚本

## 1. 配置 .env 文件
```bash
# 在项目根目录创建或编辑 .env 文件
MIMO_API_KEY=your_actual_api_key
DEEPSEEK_API_KEY=your_actual_api_key
KB_ADMIN_TOKEN=your_strong_admin_token
KB_EMBEDDER_URL=http://localhost:8081
LOADER_URL=http://127.0.0.1:8090
```

## 2. 修复模块导入路径
需要将代码中的导入路径从 `src.services.*` 改为：
- `src.services.graph_router` → `src.taiyang.graph_router`
- `src.services.wiki` → `src.taiyang.wiki`
- `src.services.distiller` → `src.shaoyang.distiller`
- `src.services.relation_builder` → `src.shaoyang.relation_builder`

## 3. 重启服务
```bash
# 停止当前服务
taskkill /PID 1532 /F

# 重新启动
cd E:\easyclaw\伏羲-v1.44\repo
python src/server.py
```

## 4. 验证修复
```bash
# 测试 API 端点
curl http://localhost:8080/api/health
curl http://localhost:8080/api/search?q=test
curl http://localhost:8080/api/system/stats
```
