**劣势**:
- 未实现真正的 CSS 模块化
- 存在样式冲突风险
- 缺乏自动化构建流程

### Vite 配置说明

```javascript
// vite.config.js
export default defineConfig({
  root: '.',
  build: {
    outDir: 'dist',
    rollupOptions: {
      input: {
        main: resolve(__dirname, 'index.html'),
        login: resolve(__dirname, 'login.html'),
      },
    },
    minify: 'terser',
  },
  server: {
    port: 3000,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true,
      },
    },
  },
});
```

**功能**:
- 多入口构建（index.html, login.html）
- API 代理配置
- Terser 压缩
- 开发服务器端口 3000
